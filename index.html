Welcome to the TCRPG Archive! This is my ongoing project to archive all the many, many wonderful stories I've had the joy of contributing to through Dungeons & Dragons--and any other Table Top Roleplaying Games, for that matter. Characters, worldbuilding, homebrew content, you name it, it'll all go here. Eventually.

This website is still in its first stage of creation. I'll have a lot more done--including this homepage--as I learn more about web development and have more free time to work on it. If you're one of the players I DM for, explore whatever's here! Nothing I put on here will contain any spoilers until it has been revealed in-game.
